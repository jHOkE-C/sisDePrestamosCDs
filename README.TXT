Este es el proyecto de Sistema de Préstamos de CDs/DVDs/Blu-ray.
------------------------------------------------------------------------
BlueJ

DESCRIPCIÓN DEL PROYECTO:
Este proyecto simula un sistema de alquiler de medios como DVDs y Blu-rays,
calculando el costo del préstamo basado en el tipo de medio, duración,
si el cliente es socio, y si se añade seguro.

CÓMO EMPEZAR:
1. Abre este proyecto en BlueJ.
2. Haz clic derecho en la clase Main y selecciona 'void main(String[] args)'.
3. Sigue las instrucciones en la terminal.
